# 因子分析（FA, Factor Analysis）

## 1. 一句话
- 假设高维观测 `x` 由少数连续因子 `z` 线性生成（再加噪声），是一种经典的“概率化降维”模型。

## 2. 定义 / 公式（线性高斯）
- 维度：`x ∈ R^d`，`z ∈ R^k`（`k << d`）
- 生成模型：
  - `z ~ N(0, I)`
  - `x = μ + W z + ε`
  - `ε ~ N(0, Ψ)`（通常 `Ψ` 取对角：每个观测维度自己的噪声方差）
- 因此边缘分布是高斯：
  - `p(x) = N(x; μ, W W^T + Ψ)`

## 3. 直觉
- `z` 捕捉“共享的变化来源”（公共因子），`ε` 捕捉“每个维度独有的噪声/细节”。
- 相比 PCA：FA 允许每个观测维度有不同噪声强度（`Ψ` 是对角而非 `σ^2 I`）。

## 4. 常用变体 / 记号差异
- Probabilistic PCA（PPCA）：`Ψ = σ^2 I` 的特例。
- 如果把 `p(x|z)` 做成非线性（神经网络），就会走向 VAE 这类模型。

## 5. 在哪些模型里出现
- 经典统计降维、作为更复杂潜变量模型的子模块。
- 与 VAE 的联系入口：[models/VAE.md](../models/VAE.md)、[modules/ELBO.md](ELBO.md)

## 6. 速查
- 关键词：loading matrix `W`，latent factors `z`，unique noise `Ψ`
- 常见坑：旋转不唯一（可解释性需要额外约束）、`W`/`Ψ` 的识别性问题

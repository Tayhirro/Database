# 诗歌收藏

> 按朝代组织存放，通过索引实现多维查询。

## 目录结构

```
诗歌收藏/
├── README.md        # 本文件
├── 索引.md          # 多维索引（按情感、主题、形式等）
├── _TEMPLATE.md     # 诗歌模板
│
├── 先秦/            # 诗经、楚辞
├── 汉/              # 汉乐府、古诗十九首
├── 魏晋南北朝/      # 建安、竹林、陶渊明、谢灵运
├── 唐/              # 初唐、盛唐、中唐、晚唐
├── 宋/              # 宋诗、宋词
├── 元/              # 元曲、元诗
├── 明/
├── 清/
└── 近现代/
```

## 组织规则

### 存放位置
```
朝代/作者/诗名.md
```
例：`唐/李白/静夜思.md`

### 标签约定（frontmatter）
每首诗使用以下标签便于多维检索：

```yaml
---
tags:
  - 朝代/唐
  - 作者/李白
  - 情感/思乡
  - 主题/月亮
  - 形式/五言绝句
---
```

### 情感/主题标签参考
- 情感：思乡、离别、相思、悲愁、豪放、闲适、感怀……
- 主题：月亮、山水、边塞、田园、咏物、咏史、节日……
- 形式：五言绝句、七言绝句、五言律诗、七言律诗、词、曲……

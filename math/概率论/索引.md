# 术语索引（概率论）

> 用法：先按关键词找；重要条目会逐步补全链接。

## B
- Borel σ-代数 `B(R^d)`：`R^d` 上的“可测集合族”，使得随机变量 `X: (Ω, F) -> (R^d, B(R^d))` 有严格定义。

## C
- 条件概率（Conditional probability）：`P(A|B) = P(A,B)/P(B)`，已知 B 发生时 A 的概率；→ [modules/ChainRule.md](modules/ChainRule.md)
- 链式法则（Chain rule / 乘法法则）：`P(A,B) = P(A|B)P(B)`，分解联合概率的核心工具；→ [modules/ChainRule.md](modules/ChainRule.md)

## D
- 分布（Distributions）：常见分布条目入口；→ [distributions/README.md](distributions/README.md)

## L
- 拉普拉斯分布 / 双指数分布（Laplace / Double Exponential）：尖峰+厚尾，常对应 `L1` 损失；→ [distributions/LaplaceDistribution.md](distributions/LaplaceDistribution.md)

## P
- 概率空间（Probability space）：`(Ω, F, P)`，其中 `F` 是 σ-代数，`P(Ω)=1`。
- 归一化（normalization）：把“非负函数”变成合法概率分布的步骤；例题见 [math/概率论/examples/Examples.md](examples/Examples.md)。

## R
- 随机变量（Random variable）：可测函数 `X: (Ω, F) -> (R^d, B(R^d))`。

## S
- σ-代数（sigma-algebra）：对补集与可数并封闭的集合族；用于定义“可测事件”。
